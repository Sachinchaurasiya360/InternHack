# InternHack

InternHack is an all-in-one SaaS platform built to support hackathon participants, organizers, mentors, and recruiters. By integrating hackathon management and career opportunities, InternHack connects coding talent directly with hiring prospects.

## Features

- **Centralized Hackathons:** Discover and join all hackathons directly on InternHackâ€”no need to visit multiple sites.
- **Internship Opportunities:** Access curated internships tailored for hackathon participants.
- **Hackathon Management:** Organize and manage events with streamlined tools.
- **Participant Dashboard:** Track your progress, achievements, and participation history.
- **Mentor Collaboration:** Connect with mentors for advice and support.
- **Recruiter Access:** Recruiters can find and engage top talent.
- **Career Integration:** Link hackathon performance to real-world job opportunities.

## Vision

InternHack strives to build a unified ecosystem where innovation meets opportunity, helping users move seamlessly from coding challenges to career advancement.

## Getting Started

InternHack is built with Next.js, Tailwind CSS, Python, Node.js, JWT, Express, Nodemailer, GPT, and Gemini.

1. Register as a participant, organizer, mentor, or recruiter.
2. Browse ongoing and upcoming hackathons.
3. Collaborate, learn, and unlock new career opportunities.
